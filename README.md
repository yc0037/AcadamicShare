# AcadamicShare

对于学生群体，在阅读论文方面可能会存在障碍，有不懂的地方，或者在阅读论文后有一些想法。我们希望开发一个学术论文讨论平台，提供笔记批注，评论以及论文推送等功能，提升论文阅读速度，增强学生间的交流。

## 开发准备

克隆仓库。

运行

``` shell
npm install
```

安装所有依赖。

运行

``` shell
npm run start
```

打开开发服务器。
